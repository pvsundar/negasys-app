## Data Files

| File | Description |
|------------------------|--------------------------------------------------|
| `TV-Utility.01b` | Conjoint part-worth utilities for 200 individuals (100 households) |
| `TV-Competition.01c` | 5 competitor TV profiles |

## Attribute Structure

| \# | Attribute | Levels |
|----------------|----------------------------------|------------------------|
| 1 | Brand Family | ValueMax, MainStreet, PrestigeView |
| 2 | Screen Size & Display | 50" 4K LED, 55" 4K QLED, 65" 4K QLED thin bezel |
| 3 | Audio Configuration | Integrated speakers, 2.1 soundbar, 5.1 system |
| 4 | Parental Controls | Basic profiles, Advanced kids mode |
| 5 | Smart Platform | Core streaming apps, Voice assistant + personalized guide |
| 6 | Price Band | Entry, Value, Premium, Ultra-premium |

## Data Format

**Utility file header:** `200,6,3,3,3,2,2,4` - 200 individuals (paired
into 100 households) - 6 attributes - Level counts: 3,3,3,2,2,4

**Competitor file header:** `5,6,3,3,3,2,2,4` - 5 competitor products -
Same attribute structure

## Case Context

Dana Briggs, Senior Category Manager at MegaMart, needs to select a smart
TV assortment for a new suburban showcase store. The target market
consists of dual-income households where both partners influence the TV
purchase decision.
